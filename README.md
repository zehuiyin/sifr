
# R Package *sifr*

Spatial Interpolation Functions in R

## Author

Zehui Yin, <zehuiyin@gmail.com>

# Installation

You can install `sifr` from [GitHub](https://github.com/) with:

``` r
if(!require(devtools)){
  install.packages("devtools")
  library(devtools)
}

devtools::install_github("zehuiyin/sifr")

library(sifr)
```
